Если программа не запустилась, то откройте папку BDE и запустите файл bdeadmin.exe. В левом окне выберите
категорию vokabul, нажмите правой кнопкой мыши и выберите в контекстном меню пункт Open. Эта категория должна выделиться 
зеленым квадратиком. Закройте окно. Поробуйте запустить программу dictonary.exe.
